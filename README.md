# How to start:

1) clone this project
2) go into the cloudwebdev directory
3) 
```
docker compose up -d
```
and wait ~ 30 seconds

4) open http://localhost:9001
    
    username: "your_username"
    password: "your_pasword" (yes, there is a typo)
5) go to the bucket "hhz"
6) click the gearwheel on the top right
7) go to "Anonymous"
8) Add Access Rule:

prefix: "/"
Access: readonly

9) save
10) Have fun!
